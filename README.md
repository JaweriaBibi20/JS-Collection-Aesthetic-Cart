# 🛒 JS Collection | Premium Minimalist Desk Store

A modern, high-class aesthetic e-commerce interface built under the brand **JS Collection**. This project showcases the power of pure semantic HTML5, clean CSS3 layouts, and core native JavaScript for seamless user interaction without relying on heavy frameworks.

## 🛠️ Tech Stack
* **HTML5:** Clean structural markup with dedicated data-attributes for dynamic state tracking.
* **CSS3:** Custom variables (CSS variables), Flexbox alignments, CSS Grid architecture, and custom keyframe animations (`slideIn`, `fadeOut`) for dynamic notifications.
* **JavaScript (ES6+):** Advanced DOM manipulation, custom array methods (`find`, `filter`), template string injection, and interactive filtering.

## ✨ Key Features
* **Dynamic Category Filtering:** Instantly sort workspace items by tech domains (Accessories, Lighting, Audio) via seamless JavaScript tab handling.
* **Live Sliding Cart Sidebar:** Toggle an aesthetic checkout panel displaying custom quantity scaling, accurate price arithmetic, and immediate remove functionalities.
* **Toast Notification Triggers:** Triggers smooth slide-in alert popups to confirm user actions ("Item added to collection!") for an immersive UX.
* **Promo Code Implementation:** Interactive validation logic that computes a live 20% total discount using promo code `JS20`.

## 📂 Project Files
* `store.html` - Premium UI brand structure.
* `store-style.css` - Soft light pastel palette setup and responsive grids.
* `store-script.js` - Dynamic computations, arrays, and event handlers.
